# Third-Party Components

This project can work with third-party PS5 debugging components:

- `libdebug.dll`
- `PS5Debug.bin`

These files are not authored by this project unless explicitly stated by their upstream maintainers. Check upstream licenses and redistribution permission before including them in public binary releases.

If you are packaging a release and are unsure about redistribution rights, remove `PS5Debug.bin` from the release ZIP and instruct users to provide their own compatible payload.
